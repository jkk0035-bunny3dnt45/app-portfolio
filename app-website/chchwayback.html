Christchurch WayBack

2020

CHCH Wayback is a high-level prototype of a 3D virtual environment that shows a timeline of the pre- and post-quake Christchurch city square. This was based on a prior demo created by the Christchurch City Council called 'Pre- and Post-Quake Visualisation of CBD.' The main features of CHCH Wayback allow a user to switch between the 2011 and 2020 versions of the city square and switch out individual buildings.


Goal of this project was to be and introduction into using the Unity SDK.
